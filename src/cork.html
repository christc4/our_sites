What is Cork?

Cork is a performant static site generator written in RC shell, leveraging the Plan9 core-utils and the simplicity of Awk

What is special about Cork?

Cork serves to be a good visualisation of directory structure, through it's sidebar function. Most static site generators bundle code that adds in support for web standards, some extra-tiny hobbyist sites with a small degitial footprint don't need all those features To see a cork powered site visit the author's!

How do I add X?

The most common user-requested feature are changes to the sidebar To extend the sidebar functionality,
  • Highlight current webpage/directories
  • Collapsible directory structure
  • Horizontal sidebar
